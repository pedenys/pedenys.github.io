Using React Native can lead web developers to troubles they don't understand.

Some ways to solve those problems can be to : 

- clear npm cache : `npm run {SCRIPT_NAME} --reset-cache`
- reinstall pods : `cd ios && rm Podfile.lock && npx pod-install`
- wipe the device (emulator)
- paste the error in StackOverflow 😅
